# Shortcus-Gallery
Personal shortcuts share

## TinyPNG

更新: 2018.11.28

使用了[TinyPNG的API](https://tinypng.com/developers)

更换apikey后使用, 可以从相册选择图片, 选择压缩后图片宽高, 也可以选择是否重命名, 最终自动上传并自动下载压缩后的图片至系统相册
